Form Validation with JavaScript

Previous Next

Build a form in an HTML file, with 3 fields email, password and confirm password, also include a submit button.
The fields must have the following validations:

  • 1 Mail must include @.
  • 2 Password must be at least 8 characters long.
  • 3 Password and confirm password must have the same value.

By clicking on the submit button, the fields must be validated, and if they are correct the border of the fields must be green, if they are incorrect, the borders must be red.
The readability of the code and good practices should be considered both in their order, the names of variables and functions.
